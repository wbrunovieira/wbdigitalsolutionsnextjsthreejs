# SEO — snapshots & plans

Periodic SEO records and the keyword strategy for wbdigitalsolutions.com.
Workflow: **research → plan → implement → re-measure → record improvement.**

## Files

- **`2026-07-11-baseline.md`** — the "photograph": full technical + GSC baseline
  on 2026-07-11, including the **zero-point positions** of every target keyword.
  Diff future snapshots against this.
- **`keyword-plan-ptbr.md`** — Portuguese / Brazil plan (4 services).
- **`keyword-plan-en.md`** — English / US-global plan.
- **`keyword-plan-it.md`** — Italian / Italy plan.
- **`keyword-plan-es.md`** — Spanish / Spain plan.

Each plan has, per service line (Websites, Automation, Platforms/Systems, AI):
a target-keyword table with intent, competition, **current GSC position**,
target page, and content action — plus a Top-8 roadmap and a tracking table.

## How positions are recorded / re-measured

Positions come from Google Search Console (service account, read-only).
To re-measure a keyword's current position:
GSC Search Analytics → dimension `query`, exact filter, current window.
Add a new dated snapshot file here and update the plans' tracking tables.

## Key finding (2026-07-11)

Mature 2-year-old site, technically excellent (Lighthouse 100/100), but
**near-zero non-brand search visibility** — 32 primary targets across 4
languages all start at "not ranking." The opportunity is topical authority in
low-competition lanes: **3D/Three.js web, n8n automation, custom AI agents/RAG**.
