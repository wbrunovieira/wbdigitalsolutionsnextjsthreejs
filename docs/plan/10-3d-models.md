## Requisitos Técnicos

### Especificações Ideais

| Aspecto | Ideal | Aceitável | Evitar |
|---------|-------|-----------|--------|
| **Polígonos** | < 20K | < 50K | > 100K |
| **Texturas** | 1K | 2K | > 4K |
| **Formato** | GLB | FBX | MAX/C4D |
| **Tamanho** | < 5MB | < 15MB | > 30MB |

### Formatos Prioritários

1. **GLB/GLTF** — Pronto para R3F, comprimido
2. **FBX** — Fácil de converter no Blender
3. **OBJ + MTL** — Funciona, pode precisar ajustar materiais
4. **Blend** — Se precisar editar

### Checklist Antes de Baixar

- [ ] Poly count está na descrição?
- [ ] Formato GLB/FBX disponível?
- [ ] Texturas incluídas?
- [ ] Preview mostra qualidade adequada?
- [ ] Estilo combina com a demo (moderno, clean)?
- [ ] Licença permite uso comercial?

## Pipeline de Otimização

### Se o modelo vier pesado:

```
1. Importar no Blender
2. Decimate Modifier (ratio 0.5)
3. Remover geometria oculta
4. Otimizar texturas (resize para 1K)
5. Export como GLB (Draco compression)
```

### Comandos Blender úteis:

```python
# Decimate
bpy.ops.object.modifier_add(type='DECIMATE')
bpy.context.object.modifiers["Decimate"].ratio = 0.5

# Export GLB
bpy.ops.export_scene.gltf(
    filepath="model.glb",
    export_format='GLB',
    export_draco_mesh_compression_enable=True
)
```

## Integração no Código

### Carregar modelo GLB

```typescript
import { useGLTF } from '@react-three/drei';

const Portal = () => {
  const { scene } = useGLTF('/models/hub/portal.glb');

  return (
    <primitive
      object={scene}
      scale={1.5}
      position={[0, 0, -10]}
    />
  );
};

// Preload para performance
useGLTF.preload('/models/hub/portal.glb');
```

### Com animações

```typescript
import { useGLTF, useAnimations } from '@react-three/drei';

const AnimatedModel = () => {
  const { scene, animations } = useGLTF('/models/portal.glb');
  const { actions } = useAnimations(animations, scene);

  useEffect(() => {
    actions['idle']?.play();
  }, [actions]);

  return <primitive object={scene} />;
};
```

### Modelo com interação

```typescript
const InteractiveProduct = () => {
  const { scene } = useGLTF('/models/product/smartphone.glb');
  const [hovered, setHovered] = useState(false);

  return (
    <primitive
      object={scene}
      scale={hovered ? 1.1 : 1}
      onPointerEnter={() => setHovered(true)}
      onPointerLeave={() => setHovered(false)}
    />
  );
};
```
